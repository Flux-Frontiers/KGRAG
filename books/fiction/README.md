# Fiction — Project Gutenberg

Place plain-text Gutenberg books in this directory.

Recommended naming: `pg<ID>_<title>.txt`
Examples:
- `pg1342_pride_and_prejudice.txt`
- `pg11_alice_in_wonderland.txt`
- `pg84_frankenstein.txt`

After adding books, run from the repo root:

```bash
kgrag init books/fiction --layer doc --name gutenberg-fiction
```
