# American Literature

American prose and poetry staged here for DocKG ingestion.

**Titles**: Moby Dick, The Scarlet Letter, Walden, Leaves of Grass,
Adventures of Huckleberry Finn, The Call of the Wild, The Red Badge of Courage,
Uncle Tom's Cabin, The Yellow Wallpaper

Run `bash books/ingest.sh /path/to/gutenberg_kg` from the repo root to populate
and build the `.dockg/` index for this genre.
