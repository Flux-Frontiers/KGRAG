# Ancient & Classical

Books from Antiquity staged here for DocKG ingestion.

**Titles**: The Odyssey, The Iliad, The Aeneid, The Republic, The Divine Comedy,
Oedipus King of Thebes, Meditations, The Bible

Run `bash books/ingest.sh /path/to/gutenberg_kg` from the repo root to populate
and build the `.dockg/` index for this genre.
