# Science — Project Gutenberg

Place plain-text Gutenberg science texts in this directory.

Recommended naming: `pg<ID>_<title>.txt`
Examples:
- `pg35_the_time_machine.txt`
- `pg164_twenty_thousand_leagues.txt`

After adding books, run from the repo root:

```bash
kgrag init books/science --layer doc --name gutenberg-science
```
