# Poetry — Project Gutenberg

Place plain-text Gutenberg poetry books in this directory.

Recommended naming: `pg<ID>_<title>.txt`
Examples:
- `pg1065_the_waste_land.txt`
- `pg23_leaves_of_grass.txt`

After adding books, run from the repo root:

```bash
kgrag init books/poetry --layer doc --name gutenberg-poetry
```
