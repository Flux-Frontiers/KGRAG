# French Literature

French prose staged here for DocKG ingestion.

**Titles**: Les Misérables, The Count of Monte Cristo, The Three Musketeers,
Madame Bovary, Twenty Thousand Leagues Under the Sea, Candide

Run `bash books/ingest.sh /path/to/gutenberg_kg` from the repo root to populate
and build the `.dockg/` index for this genre.
