# KG Family -- Brand Identity & Logo Generation Prompts

## Brand DNA

The PyCodeKG logo establishes the visual language: a dark brain silhouette filled with glowing KG nodes and domain-specific symbols, with a wordmark below in clean sans-serif. Every logo in the family follows this template -- same silhouette shape, same node-edge graph iconography, different accent color and internal symbols.

**Shared rules:**
- Brain silhouette: dark charcoal `#1A1A2E`, solid fill, faint lobe structure
- Internal graph: 6-10 glowing nodes connected by straight lines
- Node symbols: small icons encoding the domain
- Wordmark: clean Futura-style sans-serif, accent color, centered below
- Background: transparent `#0D0D1A` or transparent
- No gradients on brain, no drop shadows, no 3D effects, no lens flares
- Square composition, 1024x1024

## Generation Notes

**Recommended generators:**
- **Midjourney v6** -- append `--style raw --ar 1:1 --no photorealism, shadows, gradients` to each prompt; use the existing PyCodeKG logo as `--sref` for style consistency
- **DALL-E 3** -- prompts above work as-is; request SVG-friendly flat style in the system message
- **Stable Diffusion XL** -- use with a flat-vector LoRA; set PyCodeKG logo as style reference image

**For a cohesive set:** Generate all eight in one session using the same style reference. The KGRAG logo is the hero -- its multi-colored satellite nodes make the federation architecture self-evident at a glance.
